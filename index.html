What is MORE?

MORE is a multi-cloud tool which can be used to address many issues of cloud computing today. Multi-cloud computing is a booming industry in which customers can choose to use a combination of different cloud providers for different aspects of their business. There are two key reasons why multi-cloud should be used: avoiding the infamous vendor lock-in problem and protecting data. This cost modeling app aims to provide customers with information on which cloud providers to work with. MORE will gather customer preferences from a brief survey which will then use an algorithim to provide the best cloud services to use for a customer's needs based on data collection from the big cloud providers: AWS, Azure, and GCP.
